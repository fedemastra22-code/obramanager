Conectar ObraManager con OneDrive
Ingresá el Client ID de tu app en Azure Portal para habilitar la sincronización de datos con Excel en tu OneDrive personal.
📋 Cómo obtener el Client ID:

1. Ir a portal.azure.com con tu cuenta Microsoft
2. Buscar App registrations → New registration
3. Nombre: ObraManager · Tipos: Cuentas personales Microsoft
4. URI de redirección: ${location.href.split('?')[0]} (tipo Web)
5. Copiar el Application (client) ID y pegarlo arriba
6. En API permissions agregar: Files.ReadWrite y User.Read
